Git 上传本地文件

步骤

  1. 打开终端，进入想要的文件目录下
  2. git init //初始化git仓库
  3. git clone //克隆下载指定地址的代码（可忽略）
  4. git add . //将该目录所有东西传到缓存区
  5. git commit -m ”提交说明“ //将缓存区的东西传到本地仓库
  6. git romete add origin //添加远程版本库
  7. git push -u origin master //第一次推送 使用了-u
  8. git push origin master //以后可以用这个简便一点

报错

error: failed to push some refs to ‘https://gitee.com/***/**.git'

解决办法就是执行如下命令
git pull –rebase origin master
这个命令的就是说的 从远程仓库 clone 来的项目里包含了README.md这个文件，但是自己本地仓库里没有这个文件，于是用这个命令合并

error: src refspec master does not match any

分析：引起该错误的原因是，目录中没有文件，空目录不能提交。
